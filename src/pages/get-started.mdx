# Get Started

Welcome! This page gets you from **zero → a working `.yini` config** in a few minutes.

## 1) Install (or just use npx)

**Option A — use without installing**

```bash
npx yini --help
```

**Option B — add the CLI to your project**

```bash
npm i -D yini-cli
```

> After installing, you can run npx yini or add an npm script like "yini": "yini".

## 2) Create your first .yini file

Save this as `config.yini`:

```yini
^ App
  name = "Demo"
  version = "1.0.0"
  features = [ "search", "dark-mode" ]  # comments are allowed

^ Database
  host = "localhost"
  port = 5432
  auth = { user: "admin", pass: "secret" }  # object literal
```

Notes:

- **Sections** use `^`, add more carets for sub-sections: `^`, `^^`, `^^^`.
- **Keys** are name = value.
- **Arrays:** `[1, 2, 'three']`
- **Objects:** `{ key: 'value', n: 1 }`
- **Strings:** must be quoted `'like this'` or `"like this"`.
